# DDI Model

Documentation notes on the relation between DDI XML model and Eno model.

### Instruction

```xml
<g:ResourcePackage>
    <d:InterviewerInstructionScheme>
	    <d:Instruction>
	        <r:ID>string-id</r:ID>
		    <!-- ... -->
		</d:Instruction>
    </d:InterviewerInstructionScheme>
</g:ResourcePackage>
```

### SingleResponseQuestion

```xml
<g:ResourcePackage>
    <d:QuestionScheme>
	    <d:QuestionItem>
	        <r:ID>string-id</r:ID>
		    <!-- ... -->
		</d:QuestionItem>
    </d:QuestionScheme>
</g:ResourcePackage>
```

### MultipleResponseQuestion

```xml
<g:ResourcePackage>
    <d:QuestionScheme>
	    <d:QuestionItem>
	        <r:ID>string-id</r:ID>
		    <!-- ... -->
		</d:QuestionItem>
    </d:QuestionScheme>
</g:ResourcePackage>
```
